# Outreach Documentation Site

v0.0.1 - Job Intelligence & Market Intelligence Documentation

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy to Railway

1. Push to GitHub
2. Connect repo to Railway
3. Railway auto-detects Dockerfile
4. Deploy!

## Pages

- `/` - Dashboard & Overview
- `/todo` - TODO & Roadmap
- `/architecture` - System Architecture
- `/costs` - Cost Analysis
- `/setup` - Setup Guide
- `/profile` - Your Profile Configuration
- `/jobs` - Job Intelligence
- `/market` - Market Intelligence

## Tech Stack

- Next.js 15.1
- React 19
- Framer Motion
- Tailwind CSS
- Lucide Icons
